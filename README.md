# 概要

window.confirmのように使用できるシンプルな確認ダイアログコンポーネントです。

# デモ

[https://kamiya-kei.github.io/simple-confirm/](https://kamiya-kei.github.io/simple-confirm/)

# 使用方法

## インストール

```zsh
yarn add @kamiya-kei/simple-confirm
```

## サンプルコード
